# SortSelect

정렬 기준을 선택할 수 있는 드롭다운 컴포넌트입니다. 현재 선택된 정렬 라벨과 화살표 아이콘을 표시하며, 클릭 시 정렬 옵션 목록이 나타납니다.

## Usage

```tsx
import { SortSelect } from '@ubittz/design-system';

const options = [
  { label: '최신순', value: 'latest' },
  { label: '인기순', value: 'popular' },
  { label: '가격 낮은순', value: 'price_asc' },
];

function Example() {
  const [value, setValue] = useState<string>('latest');

  return (
    <SortSelect
      options={options}
      value={value}
      onChange={setValue}
    />
  );
}
```

## Default

값이 설정되지 않은 경우, 첫 번째 옵션의 라벨이 기본으로 표시됩니다.
