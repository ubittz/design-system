# AI Collaboration Guide

이 문서는 `@ubittz/design-system` 패키지를 사용하는 프로젝트에서 AI 어시스턴트(Claude, Cursor, Copilot 등)와 효율적으로 협업하기 위한 가이드입니다.

## 사용 방법

1. 아래에서 `AGENTS.md` 파일을 다운로드합니다
2. 프로젝트 루트에 파일을 배치합니다
3. AI 어시스턴트가 자동으로 이 파일을 참조하여 디자인 시스템 컴포넌트, 토큰, 패턴을 이해합니다
