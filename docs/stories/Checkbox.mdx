# Checkbox

체크박스 컴포넌트입니다. 약관 동의, 옵션 선택 등에 사용됩니다.

## Usage

```tsx
import { Checkbox } from '@ubittz/design-system';

function Example() {
  const [checked, setChecked] = useState(false);

  return <Checkbox label='동의합니다' checked={checked} onChange={setChecked} />;
}
```

## With Caption

## With Content

`content` prop을 전달하면 화살표 토글 버튼이 표시되고, 클릭 시 콘텐츠가 펼쳐집니다. 약관 상세보기 등에 활용됩니다.

<Canvas of={CheckboxStories.WithContent} />

## With Content Max Height

`contentMaxHeight` prop으로 펼쳐지는 콘텐츠의 최대 높이를 조절할 수 있습니다. 기본값은 80px입니다.
