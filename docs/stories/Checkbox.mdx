# Checkbox

체크박스 컴포넌트입니다. 약관 동의, 옵션 선택 등에 사용됩니다.

## Usage

```tsx
import { Checkbox } from '@ubittz/design-system';

function Example() {
  const [checked, setChecked] = useState(false);

  return (
    <Checkbox
      label="동의합니다"
      checked={checked}
      onChange={setChecked}
    />
  );
}
```
