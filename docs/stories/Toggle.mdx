# Toggle

세그먼트 형태의 토글 컴포넌트입니다. 여러 옵션 중 하나를 선택할 수 있습니다.

## Usage

```tsx
import { Toggle } from '@ubittz/design-system';

function Example() {
  const [activeIndex, setActiveIndex] = useState(0);

  return (
    <Toggle
      items={['옵션 A', '옵션 B']}
      activeIndex={activeIndex}
      onChange={setActiveIndex}
    />
  );
}
```
