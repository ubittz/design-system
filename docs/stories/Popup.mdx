# Popup

모달 형태의 팝업 컴포넌트입니다. 타이틀, 본문, 버튼을 포함합니다.

## Usage (컴포넌트 방식)

```tsx
import { Popup } from '@ubittz/design-system';

function Example() {
  const [open, setOpen] = useState(false);

  return (
    <>
      <button onClick={() => setOpen(true)}>열기</button>
      <Popup
        open={open}
        title="알림"
        body="변경사항을 저장하시겠습니까?"
        cancelButton={{ label: '취소', onClick: () => setOpen(false) }}
        confirmButton={{
          label: '확인',
          onClick: () => {
            save();
            setOpen(false);
          },
        }}
        onBackdropClick={() => setOpen(false)}
      />
    </>
  );
}
```

## Usage (Hook 방식)

```tsx
import { PopupProvider, usePopup } from '@ubittz/design-system';

// 앱 루트에 Provider 설정
function App() {
  return (
    <PopupProvider>
      <MyPage />
    </PopupProvider>
  );
}

// 컴포넌트에서 usePopup 사용
function MyPage() {
  const popup = usePopup();

  const handleDelete = () => {
    popup.open({
      title: '삭제 확인',
      body: '정말 삭제하시겠습니까?',
      cancelButton: { label: '취소' },
      confirmButton: {
        label: '삭제',
        onClick: () => deleteItem(),
      },
    });
  };

  return <button onClick={handleDelete}>삭제</button>;
}
```

### PopupButtonProps

`ButtonProps`에서 `children`을 제외하고 `label`을 추가한 타입입니다. `variant`, `size`, `disabled`, `onClick` 등 Button의 모든 props를 사용할 수 있습니다.
