# Ubittz Design System

환영합니다! Ubittz Design System은 일관된 디자인과 개발 경험을 제공하기 위한 React 기반 디자인 시스템입니다.

## 주요 특징

- **Figma 연동**: Figma Variable과 완전히 동기화된 디자인 토큰
- **플랫폼 지원**: Web과 App 플랫폼을 모두 지원
- **다국어 지원**: 한글과 영문 타이포그래피 최적화
- **테마 커스터마이징**: 브랜드 컬러를 프로젝트별로 쉽게 커스터마이징
- **Tree-shaking**: 필요한 컴포넌트만 번들에 포함

## 설치

```bash
npm install @ubittz/design-system
# or
yarn add @ubittz/design-system
```

## 빠른 시작

```tsx
import { DesignSystemProvider } from '@ubittz/design-system';

function App() {
  return (
    <DesignSystemProvider platform='web' defaultLang='kr'>
      <YourApp />
    </DesignSystemProvider>
  );
}
```

## Tailwind CSS 연동

디자인 토큰을 Tailwind CSS 유틸리티 클래스로 사용할 수 있습니다.

### Tailwind v4

CSS 진입점에서 토큰과 테마 파일을 import합니다.

```css
@import "tailwindcss";
@import "@ubittz/design-system/tokens.css";
@import "@ubittz/design-system/tailwind-theme.css";
```

`tokens.css`는 CSS 변수를 정의하고, `tailwind-theme.css`는 이를 Tailwind `@theme`에 등록하여 유틸리티 클래스를 생성합니다.

### Tailwind v3

Preset을 `tailwind.config.js`에 추가합니다.

```js
// tailwind.config.js (CommonJS)
const ubittzPreset = require('@ubittz/design-system/tailwind-preset');

module.exports = {
  presets: [ubittzPreset],
  // ...
};
```

```ts
// tailwind.config.ts (ESM)
import ubittzPreset from '@ubittz/design-system/tailwind-preset';

export default {
  presets: [ubittzPreset],
  // ...
};
```

컴포넌트를 import하면 `tokens.css`가 자동 로드되므로 별도 CSS import는 불필요합니다.
컴포넌트 없이 토큰만 사용하는 경우 CSS 진입점에 `@import "@ubittz/design-system/tokens.css"`를 추가하세요.

### 사용 예시

```html
<!-- 컬러 -->
<div class="bg-brand-primary-500 text-text-inverse">브랜드 버튼</div>

<!-- 스페이싱 -->
<div class="p-4 m-2 gap-3">레이아웃</div>

<!-- 타이포그래피 (v4: text-app-heading1 / v3: text-app-heading1) -->
<h1 class="text-app-heading1 font-kr font-semibold leading-kr-tight">제목</h1>
```

## 브랜드 컬러 커스터마이징

```tsx
<DesignSystemProvider
  platform='web'
  defaultLang='kr'
  theme={{
    brand: {
      primary: {
        500: '#FF6B00',
        600: '#CC5500',
      },
    },
  }}
>
  <YourApp />
</DesignSystemProvider>
```

## 구조

### Design Tokens

- **Colors**: Base Colors, Semantic Colors, Component Tokens
- **Typography**: Platform별, 언어별 최적화된 폰트 설정
- **Spacing**: 일관된 간격 시스템

### Components

- **Shared**: 플랫폼 공통 컴포넌트
- **App**: 모바일 앱용 컴포넌트
- **Web**: 웹용 컴포넌트
