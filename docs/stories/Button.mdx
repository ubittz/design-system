# Button

aOS 플랫폼에서 사용하는 버튼 컴포넌트 세트입니다.
5개 버튼 컴포넌트로 구성됩니다: **Button**, **IconButton**, **TextButton**, **FullButton**, **SNSButton**.

## Button

### 사용법

```tsx
import { Button } from '@ubittz/design-system';

<Button variant="primary" size="m">확인</Button>
<Button variant="ghost" shape="round" iconFront={<SearchIcon />}>검색</Button>
<Button variant="outline" size="l" fullWidth>전체 너비</Button>
<Button disabled>비활성</Button>
```

## IconButton

원형 아이콘 전용 버튼입니다. `aria-label` 필수.

```tsx
import { IconButton } from '@ubittz/design-system';

<IconButton icon={<RoundStroke.Cart size={20} />} variant="ghost" size="l" aria-label="장바구니" />
```

## FullButton

전체 너비를 차지하는 고정 높이(52px) 버튼입니다.
`FullButton.Group`으로 여러 버튼을 조합하여 사용합니다.

### 사용법

```tsx
import { FullButton } from '@ubittz/design-system';

{/* 단일 */}
<FullButton>확인</FullButton>

{/* 수직 조합 */}
<FullButton.Group layout="vertical" gap={8}>
  <FullButton variant="primary">확인</FullButton>
  <FullButton variant="ghost">취소</FullButton>
</FullButton.Group>

{/* 수평 조합 */}
<FullButton.Group layout="horizontal" gap={8}>
  <FullButton variant="ghost" style={{ flex: 1 }}>취소</FullButton>
  <FullButton variant="primary" style={{ flex: 1 }}>확인</FullButton>
</FullButton.Group>
```

## SNSButton

소셜 로그인 버튼입니다. 각 provider에 맞는 로고 SVG가 내장되어 있습니다.

### 사용법

```tsx
import { SNSButton } from '@ubittz/design-system';

{/* 원형 아이콘 */}
<SNSButton provider="kakao" />
<SNSButton provider="naver" />
<SNSButton provider="google" />
<SNSButton provider="apple" />

{/* 사각형 + 라벨 */}
<SNSButton provider="kakao" shape="square" label="카카오로 시작하기" />
```
