# TabMenu

콘텐츠 영역 내에서 탭 전환을 제공하는 52px 높이의 탭 메뉴 컴포넌트입니다.
`line`과 `box` 두 가지 스타일을 지원합니다.

## 주요 특징

- **2가지 스타일**: `line` (언더라인)과 `box` (채워진 배경) 변형 지원
- **균등 분배**: 모든 탭이 동일한 너비로 자동 분배
- **문자열 기반**: `items`에 문자열 배열만 전달하면 되는 심플한 API
- **디자인 토큰 연동**: 활성/비활성 상태의 색상이 테마에 맞게 자동 적용

## 사용법

### Line 스타일 (기본)

활성 탭 하단에 2px 보더가 표시되며, Brand Primary 색상의 텍스트가 적용됩니다.

```tsx
import { TabMenu } from '@ubittz/design-system';

function ProductList() {
  const [activeTab, setActiveTab] = useState(0);

  return (
    <>
      <TabMenu
        items={['전체', '인기', '신상품', '할인']}
        activeIndex={activeTab}
        onChange={setActiveTab}
      />
      {/* 탭에 따른 콘텐츠 렌더링 */}
    </>
  );
}
```

### Box 스타일

활성 탭에 Brand Primary 배경이 적용되며, 흰색 텍스트가 표시됩니다.

```tsx
<TabMenu
  items={['전체', '인기', '신상품', '할인']}
  activeIndex={activeTab}
  onChange={setActiveTab}
  variant="box"
/>
```

### 2탭 구성

```tsx
<TabMenu
  items={['상품 정보', '리뷰']}
  activeIndex={activeTab}
  onChange={setActiveTab}
/>
```

### Line vs Box 선택 기준

<table>
  <thead>
    <tr>
      <th>기준</th>

    </tr>
  </tbody>
</table>
