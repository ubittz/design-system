# FilterBar

가로 스크롤이 가능한 필터 버튼 바 컴포넌트입니다. 카테고리 필터링 등에 사용합니다.

## Usage

```tsx
import { FilterBar } from '@ubittz/design-system';

function Example() {
  const [selected, setSelected] = useState('all');

  const options = [
    { label: '전체', value: 'all' },
    { label: '진행중', value: 'in-progress' },
    { label: '완료', value: 'completed' },
  ];

  return (
    <FilterBar
      options={options}
      value={selected}
      onChange={setSelected}
    />
  );
}
```

## Default

기본 필터 바입니다. 필터를 클릭하면 선택 상태가 변경됩니다.
