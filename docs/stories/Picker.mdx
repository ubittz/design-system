# Picker

날짜 선택 피커 컴포넌트입니다. Calendar 컴포넌트를 팝업으로 표시하여 단일 날짜 또는 날짜 범위를 선택할 수 있습니다.

## Usage

```tsx
import { Picker } from '@ubittz/design-system';

function Example() {
  const [date, setDate] = useState<Date | null>(null);

  return (
    <Picker
      placeholder="날짜를 선택하세요"
      value={date}
      onChange={setDate}
    />
  );
}
```

## Date Range

`mode="range"`로 날짜 범위를 선택합니다.

```tsx
const [range, setRange] = useState<[Date, Date] | null>(null);

<Picker mode="range" value={range} onChange={setRange} />;
```
