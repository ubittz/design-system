# SelectableOption

선택 가능한 옵션 버튼 컴포넌트입니다. 단일 선택 또는 복수 선택 UI에 사용합니다.

## Usage

```tsx
import { SelectableOption } from '@ubittz/design-system';

function Example() {
  const [selected, setSelected] = useState(false);

  return (
    <SelectableOption
      selected={selected}
      onClick={() => setSelected(!selected)}
    >
      옵션
    </SelectableOption>
  );
}
```

## Shapes

3가지 모서리 형태를 지원합니다.

- `default` — 4px border radius
- `round` — 999px border radius (pill)
- `square` — 0px border radius
