# TextArea

여러 줄 텍스트 입력 필드 컴포넌트입니다.
`FormGroup`을 기반으로 라벨, 캡션, 에러 메시지를 지원합니다.

## 사용법

```tsx
import { TextArea } from '@ubittz/design-system';

<TextArea placeholder="내용을 입력해주세요." />
<TextArea label="내용" placeholder="내용을 입력해주세요." required />
<TextArea shape="square" placeholder="내용을 입력해주세요." />
```
