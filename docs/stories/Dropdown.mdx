# Dropdown

선택 가능한 옵션 목록을 드롭다운으로 표시하는 컴포넌트입니다.

## Usage

```tsx
import { Dropdown } from '@ubittz/design-system';

const options = [
  { label: '옵션 1', value: '1' },
  { label: '옵션 2', value: '2' },
];

function Example() {
  const [value, setValue] = useState<string>();

  return (
    <Dropdown
      placeholder="선택하세요"
      value={value}
      options={options}
      onChange={setValue}
    />
  );
}
```
