# ImageUploader

이미지 파일을 업로드하고 미리보기를 제공하는 컴포넌트입니다.

## Usage

```tsx
import { ImageUploader, ImageFile } from '@ubittz/design-system';

function Example() {
  const [files, setFiles] = useState<ImageFile[]>([]);

  return <ImageUploader value={files} onChange={setFiles} />;
}
```

## ImageFile Type

```tsx
type ImageFile = {
  id: string;      // 고유 식별자
  file?: File;     // 업로드된 File 객체 (서버 이미지는 없을 수 있음)
  url: string;     // 미리보기 URL (blob 또는 서버 URL)
  name: string;    // 파일명
};
```
