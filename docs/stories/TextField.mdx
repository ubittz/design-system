# TextField

텍스트 입력 필드 컴포넌트입니다.
`FormGroup`을 기반으로 라벨, 캡션, 에러 메시지, 버튼, 타이머 등을 지원합니다.

## TextField

### 사용법

```tsx
import { TextField } from '@ubittz/design-system';

<TextField placeholder="입력해주세요." />
<TextField label="이메일" placeholder="이메일을 입력해주세요." required />
<TextField label="금액" value="10,000" unit="원" />
<TextField shape="line" label="이름" placeholder="이름을 입력해주세요." />
<TextField label="휴대폰 번호" placeholder="010-0000-0000" format="ddd-dddd-dddd" />
```

### Format

`format` prop을 사용하면 숫자 입력 시 자동으로 포맷팅됩니다.
포맷 문자열에서 `d`는 숫자 자리를, 그 외 문자는 리터럴 구분자로 사용됩니다.

- `ddd-dddd-dddd` → `010-1234-1234`
- `ddd-dd-ddddd` → `123-45-67890`

## FormGroup

`FormGroup`은 입력 컴포넌트의 공통 래퍼입니다.
Label, Caption, Error Message, Button, Timer를 관리합니다.
TextField뿐 아니라 TextArea, Select 등에서도 재사용됩니다.

### 사용법

```tsx
import { FormGroup } from '@ubittz/design-system';

<FormGroup label="커스텀 입력" required caption="안내 문구">
  <MyCustomInput />
</FormGroup>
```

### useFormGroupProps Hook

`useFormGroupProps`는 FormGroup props와 나머지 props를 분리하는 유틸 훅입니다.
커스텀 입력 컴포넌트를 만들 때 사용합니다.

```tsx
import { FormGroup, useFormGroupProps } from '@ubittz/design-system';
import type { FormGroupProps } from '@ubittz/design-system';

type MyInputProps = React.InputHTMLAttributes<HTMLInputElement> & FormGroupProps;

function MyInput(props: MyInputProps) {
  const { formGroupProps, ...inputProps } = useFormGroupProps(props);

  return (
    <FormGroup {...formGroupProps}>
      <input {...inputProps} />
    </FormGroup>
  );
}
```
