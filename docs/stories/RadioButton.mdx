# RadioButton

라디오 버튼 컴포넌트입니다. 여러 옵션 중 하나를 선택할 때 사용합니다.

## Usage

```tsx
import { RadioButton } from '@ubittz/design-system';

function Example() {
  const [selected, setSelected] = useState<string>('option1');

  return (
    <>
      <RadioButton
        label="옵션 1"
        value="option1"
        name="my-group"
        checked={selected === 'option1'}
        onChange={() => setSelected('option1')}
      />
      <RadioButton
        label="옵션 2"
        value="option2"
        name="my-group"
        checked={selected === 'option2'}
        onChange={() => setSelected('option2')}
      />
    </>
  );
}
```

## Default

기본 라디오 버튼입니다. 클릭하여 선택/해제할 수 있습니다.

## Selected

라디오 그룹에서 하나의 옵션이 선택된 상태입니다.

<Canvas of={RadioButtonStories.Selected} />

## Size M

큰 사이즈(M)의 라디오 버튼입니다. 아이콘 24px, 텍스트 16px로 표시됩니다.

<Canvas of={RadioButtonStories.SizeM} />

## Disabled

비활성화 상태의 라디오 버튼입니다. 클릭할 수 없으며 투명도가 적용됩니다.
