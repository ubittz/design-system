# BottomSheet

하단에서 올라오는 시트 컴포넌트입니다. 다양한 컨텐츠를 `children`으로 전달합니다.

## Usage (컴포넌트 방식)

```tsx
import { BottomSheet } from '@ubittz/design-system';

function Example() {
  const [open, setOpen] = useState(false);

  return (
    <>
      <button onClick={() => setOpen(true)}>열기</button>
      <BottomSheet
        open={open}
        title="옵션 선택"
        showClose
        button={{ label: '확인', onClick: () => setOpen(false) }}
        onClose={() => setOpen(false)}
      >
        <div>커스텀 컨텐츠</div>
      </BottomSheet>
    </>
  );
}
```

## Usage (Hook 방식)

```tsx
import { BottomSheetProvider, useBottomSheet } from '@ubittz/design-system';

function App() {
  return (
    <BottomSheetProvider>
      <MyPage />
    </BottomSheetProvider>
  );
}

function MyPage() {
  const sheet = useBottomSheet();

  const handleOpen = () => {
    sheet.open({
      title: '옵션 선택',
      showClose: true,
      content: <OptionList />,
      button: {
        label: '확인',
        onClick: () => handleConfirm(),
      },
    });
  };

  return <button onClick={handleOpen}>열기</button>;
}
```
