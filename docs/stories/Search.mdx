# Search

검색 입력 필드 컴포넌트입니다.
우측에 검색 아이콘이 포함되며, Enter 키 또는 아이콘 클릭으로 검색을 실행할 수 있습니다.

## 사용법

```tsx
import { Search } from '@ubittz/design-system';

<Search placeholder="검색어를 입력하세요." />
<Search
  placeholder="검색어를 입력하세요."
  onSearch={(value) => console.log(value)}
/>
```
